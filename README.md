# 🐍 Streamlit — Guide Interactif

> Une app Streamlit pour apprendre Streamlit. Pédagogique, interactive, déployée en ligne.

**👉 [Voir l'app en live](https://streamlit-intro-badreddine.streamlit.app)** *(lien après déploiement)*

---

## 📚 Ce que tu vas apprendre

| Module | Contenu |
|---|---|
| 🎛️ Les widgets | Boutons, sliders, selectbox, text_input, checkbox, multiselect |
| 📊 Données & graphiques | DataFrames filtrables, Plotly (courbes, barres, scatter), `@st.cache_data` |
| 🧱 Layouts | Colonnes, tabs, expanders, containers, messages |
| 🚀 Déploiement | Streamlit Cloud, requirements.txt, alternatives (HF, Docker) |

---

## 🚀 Lancer localement

```bash
git clone https://github.com/BadreddineEK/streamlit-intro
cd streamlit-intro
pip install -r requirements.txt
streamlit run app.py
```
